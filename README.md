# Meeting

An example of a chat room with audio and video.
（agora rtc sdk + easemob-websdk）

# Run Dev
rename `.env.example` to `.env` , and configure the following parameters.

Start Development
```bash
npm run dev
```


# Build Prod
```bash
npm run build
```
